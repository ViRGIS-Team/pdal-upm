Property Documentation

◆ VersionInteger

int Pdal.Config.VersionInteger
get

The PDAL version number as a single integer. The version number is equal to the sum of 10,000 times the major version number, 100 times the minor version number, and the patch version number. For example, version 1.7.1 is represented as 10701.
